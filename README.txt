Salome_meca
-----------

Ce dossier contient des fichiers relatifs à l'utilisation de salome_meca (smeca).

- Workflow salome : worflow qui décrit étape par étape comment réaliser une optimisation via Python sur salome_meca;
- Calcul_parallele_smeca : Fichier exécutable tel quel qui met en oeuvre les calculs parallèles de salome_meca;
- Dossier fichiers_calculs : Dossier qui regroupe des exemples de dossiers qui sont nécessaires pour les calculs de smeca;
- Optim_composite : notebook qui montre l'optimisation dans le cadre d'un matériau composite, avec la réécriture des fichiers que cela implique;
- Optim_test_ML : implémentation d'un test d'optimsation avec l'algorithme mixte de machine learning (ML) EGO-NSGAII;
